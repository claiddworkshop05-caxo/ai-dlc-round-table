# INT-000 Intent

## Purpose
スマートフォンでQRコードを読み取ることで、社内備品（ノートPC・機器など）の貸出・返却を簡単に記録・管理できるWebアプリケーションを構築する。

## Scope
- **対象**: 社内備品の管理担当者・利用者（スマートフォン利用）
- **In Scope**:
  - 備品マスタ管理（登録・一覧・詳細）
  - 備品ごとのQRコード生成・表示
  - QRコードスキャンによる貸出・返却記録
  - 貸出履歴の閲覧
- **Out of Scope**:
  - ユーザー認証・権限管理
  - 通知・リマインダー機能
  - 備品の発注・在庫管理

## Background
現状、備品の貸出記録が紙台帳や口頭管理となっており、誰が何を借りているか把握しにくい。
QRコードを備品に貼付し、スマートフォンで読み取るだけで記録できる仕組みを作ることで、管理コストを大幅に削減する。
