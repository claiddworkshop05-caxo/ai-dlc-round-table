# INT-000 Requirements

## 機能要件

### REQ-001: 備品マスタ管理
- 備品を登録できる（名前・説明・カテゴリ）
- 備品の一覧を表示できる（ステータス付き）
- 備品の詳細を表示できる

### REQ-002: QRコード生成
- 各備品に一意のQRコードを生成・表示できる
- QRコードは備品詳細ページのURLを含む
- QRコードを印刷・保存できる

### REQ-003: QRコードスキャン
- スマートフォンのカメラでQRコードをスキャンできる
- スキャン後、該当備品の貸出/返却ページへ遷移する

### REQ-004: 貸出・返却記録
- 借りる人の名前を入力して「貸出」を記録できる
- 「返却」を記録できる
- 貸出中の備品ステータスが更新される

### REQ-005: 履歴閲覧
- 備品ごとの貸出・返却履歴を時系列で表示できる
- 全備品の貸出履歴一覧を表示できる

## 制約
- スマートフォンのブラウザで動作すること（PWA不要）
- オフライン動作は不要
- データはNeonDB（PostgreSQL）に保存
