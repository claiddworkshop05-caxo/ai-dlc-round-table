# INT-000 User Stories

## US-001: 備品登録
管理者として、備品（名前・説明・カテゴリ）を登録したい。
なぜなら、管理対象の備品をシステムに追加する必要があるから。

## US-002: QRコード表示・印刷
管理者として、備品詳細ページでQRコードを表示・印刷したい。
なぜなら、備品にQRコードシールを貼付するため。

## US-003: QRコードスキャン→貸出
利用者として、備品に貼ったQRコードをスマートフォンでスキャンして、自分の名前を入力して「貸出」を記録したい。
なぜなら、簡単に貸出申請できるようにするため。

## US-004: QRコードスキャン→返却
利用者として、QRコードをスキャンして「返却」を記録したい。
なぜなら、返却時も素早く記録できるようにするため。

## US-005: 備品一覧・ステータス確認
誰でも、備品一覧と各備品の現在の貸出状況（利用可能/貸出中）を確認したい。
なぜなら、今すぐ借りられるか確認するため。

## US-006: 貸出履歴確認
管理者として、誰がどの備品をいつ借りたか確認したい。
なぜなら、備品の利用状況を把握するため。
